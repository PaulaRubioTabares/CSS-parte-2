Color en Css

Para cambiar el color del texto de un párrafo en la hoja de estilos se coloca el selecto, dentro de ese selector se escribe la propiedad color, en donde se puede declarar este de diversas maneras

Color por nombres, color:aqua

Color por formato RGB (Red, Green, Blue), color: rgb(204, 0, 255)

Color por número hexadecimal, color: #8d1e1e

Color por formato rgba, el último número representa la intensidad, color: rgba(241, 74, 74, 0.87)

Background

Background color

Como su nombre lo indica coloca un color de fondo en el elemento

Article con fondo de color rgba(57, 132, 218, 0.596) y propiedad de opacity (opacidad de todo el elemento)

Además se añade la propiedad opacity

Background image

Coloca en el fondo del elemento una imagen

Imagen de fondo, pero al ser más grande la imagen que el texto, no se alcanza a visualizar del todo

Para poder arreglar la visualización de la imagen, se puede usar la etiqueta de background-size con sus diferentes valores, dependiendo como se quiere dejar, para demostrar su uso se va a colocar un tamaño de 200px al elemento

Background-size Auto

Imagen de fondo, elemento con un tamaño de height: 200px; e imagen con background-size: auto;
En este caso, Auto es el valor por defecto, y no realiza cambios en las proporciones de la imagen, es decir, muestra su tamaño original.

Background-size Cover

Imagen de fondo, elemento con un tamaño de height: 200px; e imagen con background-size: cover;
Cover ayuda a que la imagen cubra todo el elemento de html, además el tamaño de la imagen se va adaptando al tamaño de la ventana donde se está visualizando.

Background-size Contain

Imagen de fondo, elemento con un tamaño de height: 200px; e imagen con background-size: contain;
Contain cambia el tamaño de la imagen proporcionalmente sin distorsionarla y si la imagen con ese tamaño no cubre todo el ancho del elemento entonces duplica la imagen.

Background-size px px

Background-size también permite definir mediante pixeles el tamaño de la imagen

Imagen de fondo, elemento con un tamaño de height: 200px; e imagen con background-size: 150px 150px;
Medidas al definir tanto el ancho como el largo de la imagen esta puede distorsionar, y al igual que con contain, si la imagen con ese tamaño no cubre todo el ancho del elemento entonces duplica la imagen, y en este caso se observa que pasa si la imagen tampoco cubre el largo del elemento

Background-repeat

Para ver mejor este efecto se va a colocar un color al fondo del elemento

No-repeat

Tanto en el caso de cover, como en el definir las medidas de la imagen, se observa que la imagen se repite para cubrir todo el ancho de la página, si se quiere que esto no pase, se utiliza la propiedad de background-repeat: no-repeat;

Imagen de fondo, elemento con un tamaño de height: 300px; e imagen con background-size: 150px 150px; y background-repeat: no-repeat;

Repeat x

Pero background-repeat tiene más funciones, se puede hacer que la imagen se repita de manera horizontal a propósito con background-repeat: repeat-x;

Imagen de fondo, elemento con un tamaño de height: 300px; e imagen con background-size: 150px 150px; y background-repeat: repeat-x;

Repeat y

Y se puede hacer que la imagen se repita de manera vertical con background-repeat: repeat-y;

Imagen de fondo, elemento con un tamaño de height: 300px; e imagen con background-size: 150px 150px; y background-repeat: repeat-y;

Background-position

Para ver mejor este efecto se va a colocar un color al fondo del elemento y se va a usar background-repeat: no-repeat
Esta propiedad sirve para definir en donde va a iniciar la ubicación de la imagen, el primer valor colocado, indica el eje horizontal y el segundo indica el eje vertical

Para el eje horizontal existen los siguientes valores:

  • Left
  • Right
  • Center

Para el eje vertical existen los siguientes valores:

  • Bottom
  • top
  • Center

Eje horizontal Left, eje vertical Bottom

Imagen de fondo, elemento con un tamaño de height: 200px; imagen con background-size: 170px 170px; y background-repeat: no-repeat; y background-position: left bottom;

Eje horizontal Right, eje vertical top

Imagen de fondo, elemento con un tamaño de height: 200px; imagen con background-size: 170px 170px; y background-repeat: no-repeat; background-position: right top;

Eje horizontal Center, eje vertical Center

Imagen de fondo, elemento con un tamaño de height: 200px; imagen con background-size: 170px 170px; y background-repeat: no-repeat; background-position: center center;

Background una sola línea

Al igual que con border, muchas de las características que vimos con background se pueden colocar en una sola línea quedando: background: background-color url background-repeat background-position / background-size;
