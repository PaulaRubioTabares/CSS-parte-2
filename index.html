Color en Css

Para cambiar el color del texto de un párrafo en la hoja de estilos se coloca el selecto, dentro de ese selector se escribe la propiedad color, en donde se puede declarar este de diversas maneras

Color por nombres, color:aqua

Color por formato RGB (Red, Green, Blue), color: rgb(204, 0, 255)

Color por número hexadecimal, color: #8d1e1e

Color por formato rgba, el último número representa la intensidad, color: rgba(241, 74, 74, 0.87)

Unidades de medida

Unidades absolutas

En este caso estas unidades varían según el dispositivo en el que se estén visualizando

El tamaño de letra de este texto está en milímetros mm: font-size: 4mm

El tamaño de letra de este texto está en pulgadas in: font-size: 0.2in

El tamaño de letra de este texto está en pixeles px: font-size: 20px

El tamaño de letra de este texto está en puntos pt: font-size: 15pt

El tamaño de letra de este texto está en picas pc: font-size: 1.5pc

Al ser medidas que van a cambiar según el dispositivo en donde se está mirando la página, no se recomiendan usar mucho

Unidades relativas

Con estas unidades el tamaño del elemento va a variar relativamente en base otro elemento, ejemplo el tamaño de la fuente en html

El tamaño de letra de este texto está en rem: font-size: 2rem

Background color

Como su nombre lo indica coloca un color de fondo en el elemento

Background image

Coloca en el fondo del elemento una imagen

Imagen de fondo, pero al ser más grande la imagen que el texto, no se alcanza a visualizar del todo

Para poder arreglar la visualización de la imagen, se puede usar la etiqueta de background-size con sus diferentes valores, dependiendo como se quiere dejar, para demostrar su uso se va a colocar un tamaño de 200px al elemento
